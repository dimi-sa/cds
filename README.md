# cd on steroids

```console
foo@bar:~$ cds


docs -> /home/foo/Documents

desk -> /home/foo/Desktop

down -> /home/foo/Downloads


```

```console
foo@bar:~$ cds desk
foo@bar:~/Desktop$
```
